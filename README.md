## 🚀 التثبيت والإعداد

### 1. إعداد Google Apps Script

1. اذهب إلى [Google Apps Script](https://script.google.com/)
2. أنشئ مشروع جديد
3. انسخ محتوى ملف `apps-script.gs`
4. الصق الكود في محرر الكود
5. احفظ المشروع
6. انقر على **"نشر" → "نشر كويب تطبيق"**
7. اختر:
   - النسخة الجديدة
   - نفّذ بصفتك (حسابك الشخصي)
   - الوصول: أي شخص
8. سيظهر رابط - انسخه (مثال: `https://script.google.com/macros/d/YOUR_ID/userweb`)

### 2. تحديث رابط Web App

افتح ملف `script.js` وضع رابط Web App:

```javascript
const GOOGLE_APPS_SCRIPT_URL = 'https://script.google.com/macros/d/{YOUR_DEPLOYMENT_ID}/userweb';
```

### 3. تفعيل GitHub Pages

1. اذهب إلى إعدادات المستودع
2. اختر **Pages** من القائمة الجانبية
3. اختر:
   - **Source**: main branch
   - **Folder**: root
4. اضغط **Save**
5. سيظهر الرابط الخاص بك (مثال: `https://SOFIANETAIFI.github.io/taifishop/`)

## 🐛 استكشاف الأخطاء

### الطلبات لا تصل إلى Google Sheets
- تأكد من أن رابط Apps Script صحيح في `script.js`
- تحقق من أن الـ Web App منشور بشكل صحيح
- افتح وحدة تحكم المتصفح (F12) وتحقق من الأخطاء

### الصفحة لا تظهر بشكل صحيح
- امسح ذاكرة التخزين المؤقت (Ctrl+Shift+R)
- تأكد من أن جميع الملفات موجودة في المستودع

### الصور لا تظهر
- استخدم روابط صور كاملة (HTTPS)
- تأكد من أن الروابط صحيحة وتعمل
